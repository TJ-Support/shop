PayPal Plugin - Attribute Form Help

Attribute Settings
Option Description
Product Name Select the product to which this attribute will apply. The product must exist before you can apply attributes to it.
Attribute Name This is the name of the attribute: Size, Color, Capacity, etc. You can select an existing name, or enter a new name in the text box. Note that the names must match exactly for the attribute to be shown properly in the selection list. For example, entering "Color" for one attribute and "color" for another will result in two selections when there should probably be only one.
Attribute Value The "Attribute Value" is the specific value of the attribute. For example, if the Attribute Name is "Color", an Attribute Value might be "Blue."
Attribute Price Enter the incremental price for this specific attribute. This is the extra charge that will be applied to the product if this attribute is selected.

This value may be negative if a discount is given for this attribute.

Order The Order value indicates the order in which this Attribute Value will be displayed in the selection list. Values are automatically sequenced by multiples of 10 (10, 20, 30, etc.), so to insert a new value between two existing ones you could enter something like "25".

The attribute with the lowest order value is displayed first in the selection list, and becomes the default. You might want to make sure that an attribute with a zero price is the default.

Enabled? Check this box if this attribute value is enabled. You might want to disable a value if it is temporarily out of stock, for example.
