Gateway Configuration
Authorze.Net SIM

Option Description
Production API Login Enter the API Login ID assigned to your Production Authorize.Net account.
Production Secret Key Enter the Secret Key assigned to your Production Authorize.Net account.
Test (Sandbox) API Login Enter the API Login ID assigned to your Testing Authorize.Net account.
Test (Sandbox) Secret Key Enter the Secret Key assigned to your Testing Authorize.Net account.
URL- Production Enter the production Authorize.Net URL, e.g. "https://secure.authorize.net"
URL- Testing Enter the Authorize.Net Sandbox URL, e.g. i"https://test.authorize.net"
Production MD5 Hash Enter the MD5 hash key configured for your production Authorize.Net account.
Test MD5 Hash Enter the MD5 hash key configured for your test Authorize.Net account.
Testing (Sandbox) Mode? Check this box during testing. The Sandbox URL and other values will be used instead of Production.
Enabled Check this box to indicate that this gateway is enabled for use. If not checked, the PayPal gateway will not be used.
Buttons Select the buttons that are supported by this gateway. Payment for a shopping cart is always supported. This is primarily meant to reduce clutter in blocks and product lists, where you may want only one gateway handling immediate purchases.
Order Enter a number to represent the priority of this gateway (lower number means higher priority).
