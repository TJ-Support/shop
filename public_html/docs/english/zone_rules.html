Shop Zone Rules

Rule Name
Enter a short but descriptive name for the rule. This should be unique to tell the rules apart.
Rule Description
Enter a description for this rule. This text will be shown on the product detail page as a "restriction"
Rule Type
Select "Allowed" or "Denied" for the rule. If the type is Allowed then sales are allowed only to the regions specified in the rule. If the type is Denied then sales are allowed only to the regions NOT specified in the rule. If a region is disabled, sales are blocked to that region regardless of any rules.
Regions
Shows the global regions assigned to this rule. Regions are assigned from the "Regions" menu option. Check any regions that you wish to remove from this rule and click "Delete Selected" below. If a Region is in the rule then the rule applies to all countries and states under that region.
Countries
Shows the countries assigned to this rule. Countries are assigned from the "Countries" menu option. Check any countries that you wish to remove from this rule and click "Delete Selected" below. If a Country is shown here then this rule applies also to all states under that country.
States
Shows the states assigned to this rule. States are assigned from the "States" menu option. Check any states that you wish to remove from this rule and click "Delete Selected" below.
Enabled?
Indicate whether this rule is enabled. Disabled rules always return "OK" when checking if sales are allowed.
